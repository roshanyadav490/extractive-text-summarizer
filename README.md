# extractive-text-summarizer
Extractive Text Summarizer using Sentence Transformers (all-MiniLM-L6-v2).   Extracts the most important sentences by computing semantic similarity with the entire document – no TextRank, no frequency-based methods. Modern NLP approach with state-of-the-art sentence embeddings.

## Dataset
This project is trained/tested on publicly available datasets from Kaggle:

- **CNN/DailyMail Dataset**  
  Link: https://www.kaggle.com/datasets/gowrishankarp/newspaper-text-summarization-cnn-dailymail
